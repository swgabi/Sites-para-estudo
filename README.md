# Sites-para-estudo

Neste repositório, desenvolvo sites para estudar e praticar tudo o que estou aprendendo na matéria de programação web. 
Como não tenho clientes reais, utilizo a IA de forma honesta: ela assume o papel de um cliente fictício que contrata meus serviços, passando os requisitos, ideias e feedbacks.

Todo o código e o desenvolvimento dos sites são feitos por mim.
A IA não é usada para criar os sites, apenas para simular o processo de um projeto real.
